# Componentes SIGP · Cotizaciones

Cotizaciones en modo LPU (con snapshot de precios) o libre, versionado
ante ajustes del cliente, generación de PDF con marca NEG y consecutivo
OFR-YYYY-NNN.

Entidades: `cotizaciones` (con `modo`, `version`, `items_snapshot`,
`pdf_hash`).

Se agrega en: Fase 1 (Comercial).
